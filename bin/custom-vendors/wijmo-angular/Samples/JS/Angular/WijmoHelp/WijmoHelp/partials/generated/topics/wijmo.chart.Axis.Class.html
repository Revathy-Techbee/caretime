Properties

Name Type Description
actualMax number Gets the actual axis maximum.
actualMin number Gets the actual axis minimum.
axisLine boolean Gets or sets a value indicating whether the axis line is visible.
axisType AxisType Gets the axis type.
binding string

Gets or sets the comma-separated property names for the itemsSource property to use in axis labels.

The first name specifies the value on the axis, the second represents the corresponding axis label. The default value is 'value,text.'
format string Gets or sets the format string used for axis labels (see Globalize).
itemFormatter Function

Gets or sets the itemFormatter function for axis labels.

If specified, the function takes two parameters:

  • render engine: The IRenderEngine object to use in formatting the labels.
  • current label: A string value with the following properties:
    • value: The value of the axis label to format.
    • text: The text to use in the label.
    • pos: The position in control coordinates at which to render the label.
    • cls: The CSS class to apply to the label.

The function returns the label parameters for labels on which to modify properties.

For example: 

chart.axisY.itemFormatter = function(engine, label) {
    if(label.val > 5){
        engine.textFill = 'red'; // red text
        label.cls = null;// no default css
     }
    return label;
}

itemsSource any

Gets or sets the items source for axis labels.

The names of properties are specified by the binding.

For example:

 // default value for Axis.binding is 'value,text'
 chart.axisX.itemsSource = [ { value:1, text:'one' }, { value:2, text:'two' } ];

labelAngle number

Gets or sets the rotation angle of axis labels.

The angle is measured in degrees with valid values ranging from -90 to 90.
labels boolean Gets or sets a value indicating whether axis labels are visible.
logBase number

Gets or sets the logarithmic base of the axis.

If the base is not specified the axis uses the normal scale.
majorGrid boolean Gets or sets a value indicating whether the axis includes grid lines.
majorTickMarks TickMark Gets or sets the location of axis tick marks.
majorUnit number

Gets or sets the number of units between axis labels.

If the axis contains date values, then the units are expressed in days.
max number Gets or sets the maximum value shown on the axis. If not set, the maximum is calculated automatically.
min number Gets or sets the minimum value shown on the axis. If not set, the minimum is calculated automatically.
minorGrid boolean Gets or sets a value indicating whether the axis includes minor grid lines.
minorTickMarks TickMark Gets or sets the location of minor axis tick marks.
minorUnit number

Gets or sets the number of units between minor axis ticks.

If the axis contains date values, then the units are expressed in days.
name string Gets or sets the axis name.
origin number Gets or sets the value at which the axis crosses the perpendicular axis.
overlappingLabels OverlappingLabels Gets or sets a value indicating how to handle overlapping axis labels.
position Position Gets or sets the enumerated axis position.
reversed boolean Gets or sets a value indicating whether the axis is reversed (top to bottom or right to left).
title string Gets or sets the title text shown next to the axis.
