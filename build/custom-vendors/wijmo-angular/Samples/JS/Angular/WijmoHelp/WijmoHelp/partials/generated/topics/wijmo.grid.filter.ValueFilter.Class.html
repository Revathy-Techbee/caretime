Defines a value filter for a column on a FlexGrid control.

Value filters contain an explicit list of values that should be displayed by the grid.

Name Parameters Description

column (Column): The column to filter.

Returns: void

 Initializes a new instance of a ValueFilter.

Properties

Name Type Description
column Column Gets the Column to filter.
filterText string Gets or sets a string used to filter the list of display values.
isActive boolean

Gets a value indicating whether the filter is active.

The filter is active if at least one of the two conditions has its operator set to a non-null value.
showValues any Gets or sets an object with all the formatted values that should be shown.
